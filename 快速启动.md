## ❓ 遇到问题？

### 服务器窗口闪退？
- 说明 Python 路径有问题
- 打开命令提示符（CMD），输入：
  ```
  python3 --version
  ```
- 如果显示错误，请重新安装 Python

### 点导入没反应？
- 检查服务器窗口是否还开着
- 如果关闭了，重新运行 `RUN-SERVER.bat`

### 显示"无法连接到服务器"？
- 确认服务器窗口显示启动成功
- 重启服务器（关闭窗口，重新运行）
- 刷新网页（按F5）
