# 🏠 本地飞书表格导入使用指南

## 快速开始（3步）

### 第一步：启动服务器

双击运行：**`启动飞书解析服务.bat`**

或在命令行中运行：
```bash
python feishu-parser.py
```

看到以下提示表示启动成功：
```
╔═══════════════════════════════════════════╗
║   飞书表格解析服务器已启动                ║
║   地址: http://localhost:3000             ║
╚═══════════════════════════════════════════╝
```

### 第二步：打开网站

在浏览器中打开 **`index.html`** 文件

或访问：`http://localhost:3000` （如果服务器支持静态文件服务）

### 第三步：导入飞书表格

1. 在"当前指标缺口"区域，找到"📋 从飞书表格导入"输入框
2. 粘贴您的飞书表格链接
3. 点击 **"🚀 导入飞书表格"** 按钮
4. 等待2-3秒，看到 **"✅ 飞书表格导入成功！"**

---

## 📋 飞书表格格式要求

您的飞书表格需要包含以下列：

| 车型 | 指标类型 | 2026 Q1 | 2026 Q2 | 2026 Q3 | 2026 Q4 |
|------|---------|---------|---------|---------|---------|
| X01  | 量纲    | 100     | 120     | 150     | 130     |
| X01  | 单车维度缺口 | 500.5 | 600.8 | 700.2 | 650.3 |

**注意事项：**
- 每个车型需要2行：一行"量纲"，一行"单车维度缺口"
- 车型必须是：X01, X02, X03, X04, W01, W02, W04, W05
- 数值支持小数

## 🐛 故障排除

### 问题1：服务器无法启动

**错误：** `python: command not found`

**解决：** 安装 Python 3.x
- 下载地址：https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"

---

### 问题2：导入时显示"无法连接"

**原因：** 服务器未启动或端口被占用

**解决：**
1. 检查服务器是否正在运行
2. 查看命令行是否有错误信息
3. 确保端口 3000 未被其他程序占用

---

### 问题3：数据导入后显示不正确

**原因：** 飞书表格格式不符合要求

**解决：**
1. 检查表格是否包含必需的列
2. 确认车型名称正确（区分大小写）
3. 确保每个车型有两行（量纲 + 单车维度缺口）

---

### 问题4：浏览器报CORS错误

**原因：** 浏览器安全限制

**解决：** 使用以下方式之一
1. **推荐**：通过服务器访问（如 `http://localhost:3000`）
2. 使用 Chrome 禁用安全检查（仅开发环境）：
   ```bash
   chrome.exe --disable-web-security --user-data-dir=C:\temp\chrome
   ```
